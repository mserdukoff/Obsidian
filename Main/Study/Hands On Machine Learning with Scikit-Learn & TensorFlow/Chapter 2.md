This chapter is rather long and aims to guide the reader through an entire end-to-end machine learning project. The steps are as follows:

1. Look at the big picture
2. Get the data
3. Explore and visualize the data for insights
4. Prepare the data for machine learning algorithms
5. Select model and train
6. Fine tune the model
7. Present solution
8. Launch, monitor, maintain

### Important ML data repositories
- openml.org
- kaggle.com
- paperswithcode.com
- amazon's aws datasets registry.opendata.aws
- tensorflow datasets

and others, for more go to page 39&40.
