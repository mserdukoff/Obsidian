# Skill: Process Inbox

## Trigger phrases
"process my inbox" / "run morning capture" / "clear the inbox"

## Process
1. Read every note in 00-INBOX/
2. For each note:
   a. Determine which 01-CAPTURES subfolder it belongs to:
      - observations: something I noticed in the world, raw
      - reactions: my gut response to something I read, watched, or experienced
      - patterns: the same principle appearing in two or more domains
      - questions: something I genuinely don't know the answer to
      - numbers: a real data point with a specific figure attached
   b. Sharpen the raw note into one punchy sentence — specific enough
      that a stranger understands it with zero additional context.
      If it still needs explanation, it is not sharp enough. Rewrite it.
   c. Add exactly three tags. One must be a project tag (#grammario,
      #japanese, or another project name if present). The other two
      describe the content.
   d. Move the sharpened note to the correct subfolder with today's
      date as filename prefix: YYYY-MM-DD note-title.md
3. After processing all notes, report:
   - Total notes processed and where each one landed
   - Any patterns noticed across today's captures
   - One connection worth exploring from today's batch

## Quality bar
Vague: "Learners struggle with cases"
Sharp: "Russian B2 learners consistently misuse the prepositional case
after 'о' when the topic is abstract rather than physical"

If a note is still vague after one rewrite, flag it #needs-sharpening
and move on rather than guessing at intent.
