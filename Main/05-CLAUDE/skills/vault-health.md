# Skill: Vault Health

## Trigger phrases
"run vault health" / "audit the vault" / "optimize the vault"

## Philosophy
Never move or rename anything without showing Matt the proposed changes
first and getting explicit approval. The vault is his thinking — 
reorganizing without consent destroys context.

## Process
Run all five checks every time. Report findings per check, then present
a single consolidated action list at the end for approval.

### Check 1 — Misplaced notes
Read every note in 01-CAPTURES/ and verify it is in the correct subfolder.
A note is misplaced if:
- It contains a specific number but is not in /numbers
- It is clearly a gut reaction but is not in /reactions
- It contains a genuine unanswered question but is not in /questions
Flag each misplaced note with its current location and proposed location.

### Check 2 — Orphaned notes
Find every note in 01-CAPTURES/ and 02-CONNECTIONS/ that is not linked
from any other note and has no outgoing links.
Orphaned notes are ideas that never got connected to anything.
